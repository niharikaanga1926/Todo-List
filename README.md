# To-Do List
A simple and interactive To-Do List application that allows users to add, edit, mark as complete, and delete tasks. This project is built using HTML, CSS, and JavaScript. It's a great way to organize your tasks and stay productive.

Features:

Add new tasks to the list

Mark tasks as completed

Edit task descriptions

Delete tasks

Persistent data storage with localStorage (tasks will persist even after page refresh)

Simple and user-friendly interface

Technologies Used
HTML: Structure of the application

CSS: Styling the application

JavaScript: Logic for adding, editing, deleting, and marking tasks as completed
